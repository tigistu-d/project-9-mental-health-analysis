# Project 9: Mental Health Indicators During COVID-19

This project analyzes state-level indicators of anxiety and depressive symptoms using data from the CDC Household Pulse Survey.

## Data
- Source: CDC Household Pulse Survey
- Dataset includes aggregated state-level percentages of reported symptoms over time.

## Methods
- Data cleaning and filtering
- Exploratory analysis and visualization
- Comparative analysis across states and time

## Tools
- Python
- Pandas
- Matplotlib / Seaborn
- Google Colab

## Notes
This analysis uses aggregated survey data and does not represent individual-level diagnoses.
